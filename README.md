# 智慧廁所：AI偵測結合自動通報系統

近年來，隨著公共廁所的普及，為人們帶來便利的同時，安全與衛生的的問題也隨之出現。
為改善此議題，我們研究一套能夠提升使用體驗、增進衛生環境的智慧廁所管理系統。
透過感測技術，能夠預防導致異常發生的因素。
同時，對於管理者，我們提供了更有效率的管理方式，以提高清潔和維護效率。
本系統旨在為使用者提供安全、舒適的廁所體驗，同時為管理者提供更有效率的管理方案
